## Assembly Steps:

### 1. Create Excel Workbook
- Open Excel
- Create new workbook
- Save as "Rental_Property_Analyzer_Pro.xlsx"

### 2. Import CSVs
For each CSV file:
- Insert new worksheet
- Data → From Text/CSV → Select file
- Rename sheet (remove "_RentalAnalyzer.csv")

### 3. Format Sheets
- Format currency cells as Accounting
- Apply percentage formatting (%, ratios)
- Add borders to data entry areas
- Freeze panes on row 2 for headers

### 4. Add Conditional Formatting
- Dashboard: Color code performance metrics
- ROI Calculator: Highlight best performers
- Property Comparison: Show top rankings

### 5. Protect Formulas
- Select formula cells → Format → Lock
- Review → Protect Sheet (password: ENTax2024)
- Allow users to select unlocked cells only
